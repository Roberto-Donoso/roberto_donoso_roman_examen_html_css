Hemos estudiado tres lenguajes de programación (HTML, CSS y JS) ya que son los tres pilartes fundamentales a la hora de crear una página web. ¿Por qué? pues porque uno se utiliza para crear la estructura básica de la página web, es decir, el texto y la información que va a aparecer (Texto, imágenes, videos...) otro lo utilizamos para dar estilo por ejemplo, el color del fondo, tipo de letras, tamaño... Y por último existe otro lenguaje de estos tres comentados anteriormente que lo utilizamos para implementar animaciones y dinamismo a la página, pero... ¿Que lenguaje de programación corresponde a que? A continuación lo resumiré en una tabla para que quede todo más claro y unir conceptos.
